# IputDracin

Webstreaming drama series terbaru dan terpopuler.

## Fitur
- Streaming Drama
- Download Episode
- Subtitle Indonesia
- Update Harian

## Deployment
Project ini dideploy menggunakan Netlify dengan konfigurasi Next.js Runtime v5.
Semua dependency (`@netlify/functions`, `@netlify/ipx`, `@netlify/blobs`) telah dikonfigurasi secara manual untuk memastikan kompatibilitas penuh.
